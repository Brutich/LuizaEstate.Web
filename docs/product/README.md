# Luiza Estate — продуктовая документация

Этот раздел хранит актуальное описание продукта, принятые решения и планы развития сайта и цифровых каналов Luiza Estate.

## Назначение

Luiza Estate — персональный сервис риелтора и юриста-брокера Луизы Алиниседовой для клиентов из Москвы и Московской области. Цифровой продукт должен помогать потенциальному клиенту понять, подходит ли ему специалист, сформулировать задачу и быстро связаться удобным способом.

Сайт рассматривается не только как визитка, а как основной управляемый канал доверия и получения обращений.

## Правила ведения

1. Документация описывает продукт и бизнес-логику, а не детали реализации компонентов.
2. Существенные изменения позиционирования, структуры сайта и способов привлечения клиентов фиксируются здесь до или одновременно с разработкой.
3. Дорожная карта показывает направление и приоритеты, но не заменяет GitHub Issues для конкретных задач.
4. Неизвестные факты не выдаются за подтверждённые. Они отмечаются как гипотезы или вопросы для уточнения.
5. После принятия важного решения добавляется запись в `decisions.md`.

## Текущий статус

Текущая версия сайта является одностраничной визиткой. Ближайший этап — выполнить SEO Foundation Sprint, устранить базовые технические ограничения и затем перезапустить главную страницу как понятную посадочную страницу специалиста с сильным позиционированием, описанием услуг, доказательствами доверия и удобным сценарием обращения.
